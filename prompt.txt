# Claude CLI용 전기차 계산기 SEO 최적화 프롬프트

## 프로젝트 개요
한국어 도메인 "계산기.shop"을 이용한 전기차 보조금 및 취득세 계산기 웹사이트의 SEO 최적화 작업을 수행해주세요. 목표는 "전기차 계산기", "테슬라 계산기", "아이오닉5 계산기" 키워드에서 네이버/구글 검색 1-3위 달성입니다.

## 현재 상황
- 도메인: 계산기.shop (한국어 도메인의 SEO 우위 활용 가능)
- 현재 사이트: 단일 페이지 전기차 계산기 (index.html)
- 기능: 2024/2025년 전기차 보조금 및 취득세 실시간 계산
- 데이터: CSV 파일로 차량별/지역별 보조금 정보 관리

## 요청사항

### 1. 사이트 구조 설계
다음과 같은 페이지 구조를 제안하고 각 페이지의 URL, 메타태그, H1-H3 태그 구조를 작성해주세요:

메인 타겟 페이지:
- 메인 계산기 페이지 (/)
- 테슬라 전용 계산기 (/tesla/)
- 아이오닉5 전용 계산기 (/ioniq5/)
- EV6 전용 계산기 (/ev6/)

확장 페이지:
- 지역별 보조금 페이지 (/region/seoul/, /region/gyeonggi/ 등)
- 2025년 보조금 가이드 (/guide/subsidy-2025/)
- 전기차 비교 페이지 (/compare/)

### 2. 키워드 최적화 전략
다음 핵심 키워드들에 대한 페이지별 최적화 방안을 제시해주세요:

메인 키워드:
- 전기차 계산기
- 테슬라 계산기  
- 아이오닉5 계산기
- 전기차 보조금 계산기
- 전기차 취득세 계산

롱테일 키워드:
- 전기차 실구매가 계산
- 지역별 전기차 보조금
- 테슬라 모델Y 계산기
- 현대 아이오닉5 보조금
- 전기차 세금 계산기

### 3. 콘텐츠 최적화
각 페이지별로 다음 요소들을 최적화해주세요:

기술적 SEO:
- 메타 타이틀 (60자 이내)
- 메타 디스크립션 (160자 이내)
- 구조화된 데이터 마크업 (JSON-LD)
- 내부링크 구조

콘텐츠 전략:
- 키워드 밀도 2-3% 유지
- LSI 키워드 자연스러운 포함
- 사용자 검색 의도에 맞는 콘텐츠
- FAQ 섹션 구성

### 4. 백링크 구축 전략
다음 채널들을 통한 백링크 획득 방안을 제시해주세요:

우선순위 채널:
- 보배드림 전기차 게시판
- 네이버 카페 (전기차 동호회)
- 클리앙 자동차 소모임
- 정부 사이트 (무공해차 통합누리집)

콘텐츠 마케팅:
- 커뮤니티용 자연스러운 후기 글 작성
- 유용한 계산 가이드 제작
- 딜러/정부기관 파트너십 제안

### 5. 경쟁 분석 및 차별화
현재 전기차 계산기 시장의 주요 경쟁사들과 비교하여 차별화 포인트를 제시해주세요:

차별화 요소:
- 한국어 도메인의 SEO 우위
- 실시간 데이터 업데이트
- 지역별 세분화된 보조금 정보
- 종합적인 원스톱 계산 서비스

### 6. 성과 측정 방안
다음과 같은 KPI 달성을 위한 모니터링 방법을 제시해주세요:

목표 지표:
- "전기차 계산기" 네이버/구글 1-3위
- "테슬라 계산기" 1-5위
- "아이오닉5 계산기" 1-3위
- 월 방문자 10만명 (3개월 내)
- 평균 체류시간 3분 이상

### 7. 실행 로드맵
1주차부터 12주차까지의 단계별 실행 계획을 수립해주세요:

우선순위:
- 1-2주차: 핵심 페이지 구축 및 최적화
- 3-4주차: 백링크 구축 및 커뮤니티 활동
- 5-8주차: 콘텐츠 확장 및 지역별 페이지
- 9-12주차: 고도화 및 성과 분석

## 추가 고려사항

업계 특성:
- 전기차 정책 변화에 따른 실시간 업데이트 필요
- 보조금 소진 현황 등 시기별 이슈 대응
- 신차 출시 시 빠른 정보 반영

사용자 특성:
- 전기차 구매 고려 단계의 실질적 정보 니즈
- 정확한 계산 결과에 대한 높은 신뢰도 요구
- 간편하고 직관적인 사용성 중시

기술 제약:
- 정적 HTML/CSS/JS 기반 (서버사이드 불가)
- CSV 데이터 파일 기반 정보 관리

위 내용을 종합하여 계산기.shop이 전기차 계산기 분야에서 SEO 1위를 달성할 수 있는 구체적이고 실행 가능한 전략을 수립해주세요. 특히 한국어 도메인의 장점을 최대한 활용하고, 커뮤니티 기반 백링크 구축에 중점을 둔 실용적인 방안을 제시해주시기 바랍니다.
